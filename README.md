# kedou
 > 运用快应用 实现一个商城项目
 
 ### 安装运行
 ```
 1. git clone https://github.com/niuhuahua/kedou.git

 2. 运行`npm install` 安装依赖 建议用淘宝镜像 cnpm

 3. npm run server 
 
 4. npm run watch 

 可以安装调试工具 adb 比较方便调试项目 
 ```
 由于后端接口的原因 大部分页面还没实现 如购物车等 但项目架构已经完成 可根据自己项目情况做相应的修改
